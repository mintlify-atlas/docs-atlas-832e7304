## Overview

The Supplier API provides comprehensive endpoints for managing business partners including suppliers/vendors, customers, and members. It handles contact information, financial relationships, and permissions.

## Supplier Management

### Get Supplier Info

Retrieve detailed information about a specific supplier by ID.

```bash
GET /supplier/info?id={id}
```

<ParamField query="id" type="Long" required>
  The unique identifier of the supplier
</ParamField>

<ResponseField name="code" type="integer">
  Response status code (200 for success)
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Supplier object">
    <ResponseField name="id" type="Long">Supplier ID</ResponseField>
    <ResponseField name="supplier" type="string">Supplier/Company name</ResponseField>
    <ResponseField name="type" type="string">Type (供应商=Vendor, 客户=Customer, 会员=Member)</ResponseField>
    <ResponseField name="contacts" type="string">Contact person name</ResponseField>
    <ResponseField name="phoneNum" type="string">Mobile phone number</ResponseField>
    <ResponseField name="telephone" type="string">Landline telephone</ResponseField>
    <ResponseField name="email" type="string">Email address</ResponseField>
    <ResponseField name="fax" type="string">Fax number</ResponseField>
    <ResponseField name="address" type="string">Physical address</ResponseField>
    <ResponseField name="taxNum" type="string">Tax registration number</ResponseField>
    <ResponseField name="bankName" type="string">Bank name</ResponseField>
    <ResponseField name="accountNumber" type="string">Bank account number</ResponseField>
    <ResponseField name="taxRate" type="BigDecimal">Tax rate</ResponseField>
    <ResponseField name="enabled" type="boolean">Whether supplier is enabled</ResponseField>
    <ResponseField name="advanceIn" type="BigDecimal">Advance payment (for members)</ResponseField>
    <ResponseField name="beginNeedGet" type="BigDecimal">Beginning receivables</ResponseField>
    <ResponseField name="beginNeedPay" type="BigDecimal">Beginning payables</ResponseField>
    <ResponseField name="allNeedGet" type="BigDecimal">Total receivables</ResponseField>
    <ResponseField name="allNeedPay" type="BigDecimal">Total payables</ResponseField>
    <ResponseField name="description" type="string">Description</ResponseField>
  </Expandable>
</ResponseField>

### List Suppliers

Retrieve a paginated list of suppliers with optional search filters.

```bash
GET /supplier/list?search={search}
```

<ParamField query="search" type="string">
  JSON string containing search parameters:
  - `supplier`: Supplier name filter
  - `type`: Type filter (供应商/客户/会员)
  - `contacts`: Contact person filter
  - `phonenum`: Phone number filter
  - `telephone`: Telephone filter
</ParamField>

### Create Supplier

Create a new supplier, customer, or member.

```bash
POST /supplier/add
```

<ParamField body="supplier" type="string" required>
  Supplier/Company name
</ParamField>

<ParamField body="type" type="string" required>
  Type: "供应商" (Vendor), "客户" (Customer), or "会员" (Member)
</ParamField>

<ParamField body="contacts" type="string">
  Contact person name
</ParamField>

<ParamField body="phoneNum" type="string">
  Mobile phone number
</ParamField>

<ParamField body="telephone" type="string">
  Landline telephone
</ParamField>

<ParamField body="email" type="string">
  Email address
</ParamField>

<ParamField body="address" type="string">
  Physical address
</ParamField>

<ParamField body="taxNum" type="string">
  Tax registration number
</ParamField>

<ParamField body="bankName" type="string">
  Bank name
</ParamField>

<ParamField body="accountNumber" type="string">
  Bank account number
</ParamField>

<ParamField body="enabled" type="boolean">
  Enable status (default: true)
</ParamField>

<CodeGroup>
```bash cURL
curl -X POST "http://localhost:9999/jshERP-boot/supplier/add" \
  -H "Content-Type: application/json" \
  -d '{
    "supplier": "ABC Trading Co.",
    "type": "供应商",
    "contacts": "John Smith",
    "phoneNum": "13800138000",
    "email": "john@abctrading.com",
    "address": "123 Business St, Shanghai",
    "enabled": true
  }'
```

```json Response
{
  "code": 200,
  "data": {
    "message": "操作成功"
  }
}
```
</CodeGroup>

```bash

```

<ParamField query="ids" type="string" required>
  Comma-separated supplier IDs (e.g., "1,2,3")
</ParamField>


</ParamField>

</ParamField>

Search for customers (type="客户") for dropdown selection.

```bash

```

<ParamField body="key" type="string">
  Search keyword
</ParamField>

<ParamField body="organId" type="Long">
  Filter by organization ID
</ParamField>

<ParamField body="limit" type="integer">
  Maximum number of results
</ParamField>

<ResponseField name="data" type="array">
  <Expandable title="Customer items">
    <ResponseField name="id" type="Long">Customer ID</ResponseField>
    <ResponseField name="supplier" type="string">Customer name</ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST "http://localhost:9999/jshERP-boot/supplier/findBySelect_cus" \
  -H "Content-Type: application/json" \
  -d '{
    "key": "ABC",
    "limit": 20
  }'
```

```json Response
[
  {
    "id": 5,
    "supplier": "ABC Corp"
  },
  {
    "id": 12,
    "supplier": "ABC Trading"
  }
]
```
</CodeGroup>
