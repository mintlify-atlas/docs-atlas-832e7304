## Overview

The Material API provides comprehensive endpoints for managing products, materials, inventory items, and their associated information including pricing, units, categories, and stock levels.

## Material Management

### Get Material Info

Retrieve detailed information about a specific material by ID.

```bash
GET /material/info?id={id}
```

<ParamField query="id" type="Long" required>
  The unique identifier of the material
</ParamField>

<ResponseField name="code" type="integer">
  Response status code (200 for success)
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Material object">
    <ResponseField name="id" type="Long">Material ID</ResponseField>
    <ResponseField name="name" type="string">Material name</ResponseField>
    <ResponseField name="categoryId" type="Long">Category ID</ResponseField>
    <ResponseField name="model" type="string">Model number</ResponseField>
    <ResponseField name="standard" type="string">Product standard</ResponseField>
    <ResponseField name="color" type="string">Color</ResponseField>
    <ResponseField name="brand" type="string">Brand name</ResponseField>
    <ResponseField name="mfrs" type="string">Manufacturer</ResponseField>
    <ResponseField name="unit" type="string">Unit of measure</ResponseField>
    <ResponseField name="unitId" type="Long">Unit ID (for multi-unit)</ResponseField>
    <ResponseField name="weight" type="BigDecimal">Weight</ResponseField>
    <ResponseField name="expiryNum" type="integer">Expiry days</ResponseField>
    <ResponseField name="enabled" type="boolean">Whether material is enabled</ResponseField>
    <ResponseField name="enableSerialNumber" type="string">Serial number tracking ("1" enabled)</ResponseField>
    <ResponseField name="enableBatchNumber" type="string">Batch number tracking ("1" enabled)</ResponseField>
    <ResponseField name="position" type="string">Storage position</ResponseField>
    <ResponseField name="otherField1" type="string">Custom field 1</ResponseField>
    <ResponseField name="otherField2" type="string">Custom field 2</ResponseField>
    <ResponseField name="otherField3" type="string">Custom field 3</ResponseField>
    <ResponseField name="remark" type="string">Additional remarks</ResponseField>
  </Expandable>
</ResponseField>

### List Materials

Retrieve a paginated list of materials with optional search filters.

```bash
GET /material/list?search={search}
```

<ParamField query="search" type="string">
  JSON string containing search parameters:
  - `categoryId`: Category filter
  - `materialParam`: Name/barcode search
  - `standard`: Standard specification
  - `model`: Model number
  - `color`: Color filter
  - `brand`: Brand filter
  - `mfrs`: Manufacturer filter
  - `enabled`: Enabled status
  - `enableSerialNumber`: Serial tracking filter
  - `enableBatchNumber`: Batch tracking filter
</ParamField>

### Create Material

Create a new material/product.

```bash
POST /material/add
```

<ParamField body="name" type="string" required>
  Material name
</ParamField>

<ParamField body="categoryId" type="Long" required>
  Category ID
</ParamField>

<ParamField body="unit" type="string" required>
  Unit of measure
</ParamField>

<ParamField body="model" type="string">
  Model number
</ParamField>

<ParamField body="standard" type="string">
  Product standard
</ParamField>

<ParamField body="color" type="string">
  Color
</ParamField>

<ParamField body="brand" type="string">
  Brand name
</ParamField>

<ParamField body="mfrs" type="string">
  Manufacturer
</ParamField>

<ParamField body="weight" type="BigDecimal">
  Product weight
</ParamField>

<ParamField body="enabled" type="boolean">
  Enable status (default: true)
</ParamField>

<ParamField body="enableSerialNumber" type="string">
  Enable serial number tracking ("1" for yes)
</ParamField>

<ParamField body="enableBatchNumber" type="string">
  Enable batch number tracking ("1" for yes)
</ParamField>

<CodeGroup>
```bash cURL
curl -X POST "http://localhost:8080/material/add" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Laptop Computer",
    "categoryId": 5,
    "model": "XPS-15",
    "brand": "Dell",
    "unit": "台",
    "standard": "15.6inch",
    "enabled": true
  }'
```

```json Response
{
  "code": 200,
  "data": {
    "message": "操作成功"
  }
}
```
</CodeGroup>

## Material Search & Selection

### Find by Select

Search materials for dropdown/select components with pagination.

```bash
GET /material/findBySelect
```

<ParamField query="page" type="integer" required>
  Page number (1-based)
</ParamField>

<ParamField query="rows" type="integer" required>
  Number of rows per page
</ParamField>

<ParamField query="q" type="string">
  Search query (name, barcode, etc.)
</ParamField>

<ParamField query="categoryId" type="Long">
  Filter by category
</ParamField>

<ParamField query="depotId" type="Long">
  Include stock info for depot
</ParamField>

<ParamField query="standardOrModel" type="string">
  Filter by standard or model
</ParamField>

<ParamField query="color" type="string">
  Filter by color
</ParamField>

<ParamField query="brand" type="string">
  Filter by brand
</ParamField>

<ParamField query="enableSerialNumber" type="string">
  Filter serial-tracked items
</ParamField>

<ParamField query="enableBatchNumber" type="string">
  Filter batch-tracked items
</ParamField>

<ResponseField name="total" type="integer">
  Total matching records
</ResponseField>

<ResponseField name="rows" type="array">
  <Expandable title="Material items">
    <ResponseField name="id" type="Long">Material extend ID</ResponseField>
    <ResponseField name="mBarCode" type="string">Barcode</ResponseField>
    <ResponseField name="name" type="string">Material name</ResponseField>
    <ResponseField name="model" type="string">Model</ResponseField>
    <ResponseField name="standard" type="string">Standard</ResponseField>
    <ResponseField name="unit" type="string">Unit with ratio info</ResponseField>
    <ResponseField name="stock" type="BigDecimal">Current stock</ResponseField>
    <ResponseField name="sku" type="string">SKU code</ResponseField>
    <ResponseField name="enableSerialNumber" type="string">Serial tracking enabled</ResponseField>
    <ResponseField name="enableBatchNumber" type="string">Batch tracking enabled</ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:8080/material/findBySelect?page=1&rows=20&q=laptop&depotId=1"
```

```json Response
{
  "total": 45,
  "rows": [
    {
      "id": 101,
      "mBarCode": "LAP001",
      "name": "Dell Laptop",
      "model": "XPS-15",
      "standard": "15.6inch",
      "unit": "台[基本]",
      "stock": 25,
      "brand": "Dell",
      "enableSerialNumber": "1"
    }
  ]
}
```
</CodeGroup>

### Get Material by Barcode

Retrieve material information by barcode with pricing and stock.

```bash
GET /material/getMaterialByBarCode
```

<ParamField query="barCode" type="string" required>
  Material barcode
</ParamField>

<ParamField query="depotId" type="Long">
  Depot ID for stock info
</ParamField>

<ParamField query="organId" type="Long">
  Organization ID for pricing
</ParamField>

<ParamField query="mpList" type="string" required>
  Material property list (comma-separated)
</ParamField>

<ParamField query="prefixNo" type="string">
  Bill prefix (CGDD, XSCK, etc.) for price selection
</ParamField>

<ResponseField name="code" type="integer">
  Response status code
</ResponseField>

<ResponseField name="data" type="array">
  <Expandable title="Material with pricing">
    <ResponseField name="id" type="Long">Material ID</ResponseField>
    <ResponseField name="meId" type="Long">Material extend ID</ResponseField>
    <ResponseField name="name" type="string">Material name</ResponseField>
    <ResponseField name="barCode" type="string">Barcode</ResponseField>
    <ResponseField name="stock" type="BigDecimal">Current stock</ResponseField>
    <ResponseField name="billPrice" type="BigDecimal">Bill price (based on prefixNo)</ResponseField>
    <ResponseField name="purchaseDecimal" type="BigDecimal">Purchase price</ResponseField>
    <ResponseField name="wholesaleDecimal" type="BigDecimal">Wholesale price</ResponseField>
    <ResponseField name="commodityDecimal" type="BigDecimal">Retail price</ResponseField>
    <ResponseField name="depotId" type="Long">Default depot ID</ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:8080/material/getMaterialByBarCode?barCode=LAP001&depotId=1&mpList=model,brand&prefixNo=XSCK"
```

```json Response
{
  "code": 200,
  "data": [
    {
      "id": 50,
      "meId": 101,
      "name": "Dell Laptop XPS-15",
      "barCode": "LAP001",
      "stock": 25,
      "billPrice": 8500.00,
      "purchaseDecimal": 7000.00,
      "wholesaleDecimal": 8500.00,
      "commodityDecimal": 9200.00,
      "depotId": 1,
      "unit": "台"
    }
  ]
}
```
</CodeGroup>
